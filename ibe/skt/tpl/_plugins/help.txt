Este repositório contém os plugins do aplicativo. Os plugins inseridos aqui podem
ser vistos em todo o aplicativo. Caso deseje criar um plugin específico a algum
módulo, terá de se criar um repositório como este dentro do repositório específico
do módulo. Todas as regras valem para tanto os repositórios de todo o aplicativo
quanto para do módulo específico.

Um plugin não pode ser instanciado chamando-se o operador new. A instanciação
deve ser realizada através do criador de plugins da seguinte forma;

    Ibe_Plugin::get('NomeDoPlugin');

As regras para a criação de um plugin são as seguintes:
 - O plugin deve extender a classe Ibe_Plugin
 - O plugin deve, obrigatóriamente, implementar o método dispatch
 - o método dispatch deve conter a lógica do plugin

A lógica interna dos plugins é:
 - É requisitado a instanciação do plugin através do método Get de Ibe_Plugin
 - Verifica-se se o plugin existe no núcleo do framework
    - Se não existr verifica se o plugin existe no repositório do aplicativo
    - se não existir verifica se o plugin existe no repositório do módulo atual
- Caso exista o plugin em algum destes casos,há a verificação se o mesmo é filho
da classe Ibe_Plugin, se tudo estiver ok, é retornado um objeto do Tipo Ibe_Plugin
